Motivations

Our purpose is to help people easily identify different cat breeds. It can help people with allergies to see what cats are safe and what cats are not. Our program identifies a cat’s breed from 7 options, including hypoallergenic cats, seeing if it’s safe for a person with allergies, or unsafe.

Data

Dataset


There are 7 classes. The total number of images is 5143. 80% of the total will be in the training set (AI will see the image with the correct label), and 20% of the total images will be in the test set (AI will only see the image, use the algorithm to predict the label. We compare the predicted values with the true values to determine how well the algorithm works). 10% of the training goes to the validation set (AI sees both images and their correct labels).

Data Augmentation



In deep learning, it is important to have an ample amount of data. Without enough data, you might get inaccurate results, which you obviously do not want or need. But say that, for some reason, you don’t have enough data. What do you do? Enter data augmentation! With data augmentation, you can translate, crop, scale, and flip an image, to name a few. That makes more images. With more images, you get more accurate results. Translating an image is basically just moving an image horizontally and/or vertically. Cropping an image is basically cutting out a portion of an image, deleting the rest of the image, and making that portion the same size as the original image. Next is scaling an image. There are two ways to scale an image; outwards and inwards. Outwards scaling enlarges an image, then only keeps a portion the size of the original picture (from the center of the enlarged image). Inward scaling reduces the size of an image. Flipping is what it sounds like. You can flip vertically or horizontally.

Algorithms 

Convolutional Neural Network 


CNN architecture is made by a stack of layers that make an input volume into an output volume into a function. Not many layers are used though. CNN Inception works by showing a convolution with 3 different filters. The filters are 1x1, 3x3, and 5x5. There’s also something called max pooling. Max pooling is something that finds the largest value in each part of the convolution. Max pooling is used in CNN, then by making the CNN perform its convolutions on the same type, the network gets wider. To make the process less expensive, there's an extra 1x1 added before the 3x3 and 5x5 layers. Now the 1x1 are a lot cheaper than 5x5 convolutions. 

Inception Neural Network 

Inception v3 was proposed in the paper Rethinking the Inception Architecture for Computer Vision, published in 2015. In comparison to other CNNs, Inception Networks (GoogLeNet/Inception v1) have proved to be more computationally efficient, both in terms of the number of parameters generated by the network and the economical cost incurred (memory and other resources). The techniques include factorized convolutions, regularization, dimension reduction, and parallelized computations. 

Inception v3 is a CNN with transferred learning, which means inception v3 was a pre-trained model. It was originally trained on 1000 classes, so we only need to tweak the last few layers so that it can be applied to our dataset. For instance, the right figure above shows that we modified the inception v3 model to classify 7 cat breeds. 

Evalution of Inception v3

Accuracy  - If you are accurate you are doing something correctly and very precise. The more accurate you are the more precise you are.
Precision  - If you are precise with something that means you are exactly on point and are very accurate.
Recall  - Recall calculates how many of the actual positives are true by labeling it as a true positive.
F1  - F1 is a function of Precision and Recall
Confusion Matrix  - Confusion Matrix is a summary of prediction results on a classification problem.
These are all important words to know the meanings of before exploring the topic of Evaluation and Comparison of Models.

The best accuracy for the training set occurs at epoch 10, at which point the accuracy is 91.83 and the loss is 0.2916. The accuracy represents the measure of performance of the model in terms of what percent of the predictions are correct. On the other hand, loss function is a measure of how well they fit it in terms of differences in the predictive value vs. the true value. The goal was to minimize the loss function and maximize the accuracy.  In terms of meeting the project objectives, accuracy is more important as that reflects the predictive power of the model, so we select the solution with the highest accuracy even though it is not the lowest loss function it is close to the lowest value of the loss function in Epoch 14 of 0.2712. We ran the model 15 times and the best time was the 10th one. 

The accuracy for the test set is 89.36%.
If 90% accuracy for the training set, but 60% accuracy for the test set, your data is overfitting. The algorithm does not learn enough data. However, for this model, the accuracy for both training and test sets are about 90% accuracy, which shows that the model performs well. 

Since there are 7 different classes of the cat breeds the model is trained to predict the right breed for each row of the test dataset. The confusion matrix provides the performance of the model in terms of its ability to predict the right breed of the cat. So the rows of the confusion matrix tell you the actual breed of each of the cats while the column provides the predicted breed of that cat. For a perfect model the confusion matrix will be a diagonal matrix, and the non diagonal elements will be zero. As an example, for the Bengal cat (second row) there were a total of 419 such cats in the dataset and the model identified 399 out of those correctly and 20 incorrectly so the accuracy of the model is 399/419 which is 95%.
